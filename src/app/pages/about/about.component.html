Project info

Calendar web application. Register an user and create, edit or delete events your own events.

Is possible to edit the event by dragging or resizing it in week and day view. Events can take 1 day or more. You can view by month, week or day view.

Technologies

The project is builded with Angular 11, Bootstrap 4 and SCSS preprocessor for styles.
